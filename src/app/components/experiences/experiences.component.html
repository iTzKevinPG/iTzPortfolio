Riskment Suite !

Riskment Suite fue una aplicación web desarrollada para la gestión de riesgos, generalmente usada por bancos.

Participe en el desarrollo de nuevas funcionalidades, migraciones y actualizaciones de la aplicación. La aplicación era un monolito el cual entendía todas las características necesarias para cumplir con las necesidades de sus clientes.

Pirani Risk !

Pirani Risk es una aplicación web desarrollada para la gestión de riesgos, siendo la evolución de Riskment Suite totalmente cloud, esta teniendo como objetivo la democratización de la gestión de riesgos.

Participe en el desarrollo de nuevas funcionalidades y actualizaciones de la aplicación. La aplicación aplica micro servicios y micro front-eds el cual se enfoca en el soporte de procesos grandes y diferentes módulos con características dinámicas, contando con una experiencia de usuario muy agradable y amigable.
