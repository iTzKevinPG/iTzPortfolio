Sismicidad global !

Este es un proyecto que expone mediante una tabla los datos sísmicos del ultimo mes de todo el mundo, contiene un filtro para el tipo de sismo y su paginado.

En este proyecto pude implementar mis habilidades con el entendimiento de APIs, conceptos de micro servicios, capacidad para manejar el Back-End y Front-End por separado, con el fin de aprender y fortalecer mis conocimientos sobre Ruby, React js y sus frameworks.

Glamping !

Este es un proyecto que expone un poco de información sobre un glamping cercano a el Peñol Antioquia, es una propuesta presentada como proyecto de semestre que aparte de obtener conocimientos, es un avance a un futuro proyecto que se usara por dicho glamping.

En este proyecto fui uno de los participantes que dio guía y pie a como íbamos a diseñar, desarrollar y abordar el proyecto. Junto a mis compañeros, logramos implementar un Front-End con React js y un Back-End con Go Lang y unos diseños en Figma. Este proyecto es una primer propuesta para ver su viabilidad.
